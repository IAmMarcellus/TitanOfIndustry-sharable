# Redacted Agent Org

This directory preserves the version-controlled agent roster and reporting structure for interview review.

Production company names, live agent IDs, business context, capital amounts, live-mode details, host paths, tool wiring, credentials, and operating instructions have been replaced with `<REDACTED>` placeholders.

See `MANIFEST.tsv` for the sanitized org map and the `*.AGENTS.md` files for redacted prompt placeholders.
